The website includes sign-in and sign-up forms for user authentication, enabling personalized card management. 03

Number Game

I created a fun and interactive Number Generation Game using HTML, CSS, and JavaScript. The game generates a set of random numbers and gives players a score based on specific conditions: If the numbers have a difference of 1, the player wins a large reward. If all the numbers are even, the player wins a moderate reward. If neither condition is met, a message is displayed indicating the outcome. Each time the player clicks "Play," the game generates a new set of random numbers, updates the score, and shows the remaining balance. This project helped me enhance my JavaScript skills, particularly in random number generation, conditional logic, and creating dynamic, interactive experiences for users.

04

Raw NTR Clone Website

rawntrclone is a web application that replicates essential services for blood donation, food ordering, and membership management. Built with HTML, CSS, and JavaScript, the project offers interactive features such as blood donor requests, membership sign-up, and food ordering. User data from forms is stored in localStorage, allowing information to be saved and accessed across sessions. Designed with a responsive layout, this application ensures accessibility and a seamless user experience across devices.
